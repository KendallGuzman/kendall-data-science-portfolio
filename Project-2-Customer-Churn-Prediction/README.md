# Customer Churn Prediction

This project analyzes customer churn for a telecommunications company.

The objective is to identify patterns in customer behavior and build a machine learning model capable of predicting which customers are likely to leave the service.

## Dataset
Telco Customer Churn dataset.

## Tools
Python
Pandas
Machine Learning (Scikit-Learn)

## Project Steps
1. Data inspection
2. Data cleaning
3. Exploratory analysis
4. Feature engineering
5. Machine learning model
6. Model evaluation

## Data Inspection

The first step of the analysis was to inspect the dataset structure and understand the available variables before performing any cleaning or modeling.

This inspection included:
- previewing the first rows of the dataset
- checking the number of rows and columns
- reviewing all column names
- identifying data types and non-null counts

This step is essential in data science because it helps detect missing values, understand variable types, and identify which features may be useful for prediction.
